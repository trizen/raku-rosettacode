# [Catalan numbers][1]





The recursive formulas are easily written into a constant array, either:

```perl
constant Catalan = 1, { [+] @_ Z* @_.reverse } ... *;
```


or

```perl
constant Catalan = 1, |[\*] (2, 6 ... *) Z/ 2 .. *;


# In both cases, the sixteen first values can be seen with:
.say for Catalan[^16];
```
