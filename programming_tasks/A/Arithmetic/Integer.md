# [Arithmetic/Integer][1]





Note that `div` **requires** integer arguments. If you want integer division with other types, say `floor($a/$b)`.

```perl
my Int $a = get.floor;
my Int $b = get.floor;

say 'sum:              ', $a + $b;
say 'difference:       ', $a - $b;
say 'product:          ', $a * $b;
say 'integer quotient: ', $a div $b;

say 'exponentiation:   ', $a**$b;
```
