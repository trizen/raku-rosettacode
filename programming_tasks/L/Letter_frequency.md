# [Letter frequency][1]

In perl6, whenever you want to count things in a collection, the rule of thumb is to use the Bag structure.

```raku
.&ws.say for slurp.comb.Bag.sort: -*.value;
 
sub ws ($pair) { 
    $pair.key ~~ /\n/
    ?? ('NEW LINE' => $pair.value)
    !! $pair.key ~~ /\s/
    ?? ($pair.key.uniname => $pair.value)
    !! $pair
}
```
